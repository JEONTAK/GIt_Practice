[개인]
1. git pull origin dev
2. git checkout -b 이름
3. 해당 브랜치에 본인 디렉토리 생성 후 해당 디렉토리에서 코드 작성
4. git add, commit, push
   4-1. git add .
   4-2. git commit -m "커밋 내용"
   4-3. git push origin 본인 브랜치 이름
5. github에서 Compare & Pull Request 수행
6. 1 -> 3 -> 4 -> 5번 반복
